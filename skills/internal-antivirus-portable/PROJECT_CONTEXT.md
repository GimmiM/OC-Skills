# Internal Antivirus Portable — Project Context

## Default mode
- Delivery mode: local
- No external chat lock required out of the box.

## Optional environment setup
Run setup wizard if delivery lock is needed for your environment:

```bash
python3 scripts/setup_wizard.py
```

## Command aliases
- `/av help`
- `/av setup`
- `/av scan <path|url>`
- `/av install <path> -- <install_cmd>`
- `/av surface`
- `/av policy-check`
- `/av regression`
- `/av rotate-audit`
- `/av report`
